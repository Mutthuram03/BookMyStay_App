# 🏨 Book My Stay App

### Hotel Booking Management System using Core Java & Data Structures

## 📌 Project Overview

**Book My Stay App** is a console-based **Hotel Booking Management System** developed using **Core Java and fundamental Data Structures**. The project demonstrates how real-world software systems can be built step-by-step by applying programming concepts and structured problem solving.

The system is developed **incrementally using multiple use cases**, where each use case introduces a new functionality and concept such as application startup, booking management, inventory handling, and prevention of double booking.

This project focuses primarily on **core logic, system design, and data structure usage** rather than graphical user interface development.

---

# 🎯 Objectives

The main objectives of this project are:

* Demonstrate the **practical use of Core Java concepts**
* Apply **data structures in real-world scenarios**
* Understand **application flow and system behavior**
* Implement a **modular and scalable system**
* Prevent common issues like **double booking and inconsistent inventory**
